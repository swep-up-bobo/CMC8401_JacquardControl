# CMC8401 针刺提花机控制系统仿真

基于 Qt6 Widgets 的桌面仿真应用程序，模拟工业触摸屏 HMI 界面，用于针刺提花机控制系统的教学演示与功能验证。

---

## 功能特性

- 6 个独立功能页面，覆盖标准生产、间歇模式、渐变模式、手动调试、厂家设置、报警复位全部工况
- 右侧导航面板一键切换，当前活动页面高亮指示
- 顶部状态栏实时显示运行状态与系统时间（每秒刷新）
- 底部状态栏持续监控上/下压板位置及压板间隔
- 页面间交叉导航（报警复位 ↔ 厂家设置）
- 按钮颜色语义化：开始(绿)、停止(红)、写入/修改(青)
- 全局 QSS 深色工业主题，集中管理，易于定制
- 窗口分辨率 800×480，模拟真实工业触摸屏尺寸

## 安装与构建

### 1. 克隆或下载项目

```bash
cd CMC8401_JacquardControl
```

### 2. 配置与编译

```bash
cmake -B build
cmake --build build
```

> 如果 CMake 找不到 Qt6，可手动指定路径：
> ```bash
> cmake -B build -DCMAKE_PREFIX_PATH=/opt/homebrew/opt/qt@6
> ```

### 3. 运行

```bash
./build/CMC8401_JacquardControl
```

程序启动后将显示 800×480 的仿真界面，默认进入标准模式页。

## 界面架构

整个应用分为四大区域：

```
┌─────────────────────────────────────────────────┐
│  TitleBar  │  待机中||          2026-03-16 20:00 │
├──────────────────────────────┬──────────────────┤
│                              │   [标准模式] ◄─  │
│                              │   [间歇模式]     │
│    QStackedWidget            │   [渐变模式]     │
│    (6 个功能页面)             │   [手动调试]     │
│                              │   [厂家设置]     │
│                              │   [报警复位]     │
├──────────────────────────────┴──────────────────┤
```

| 区域           | 组件              | 说明                                 |

| 底部状态栏     | `StatusBar`       | 上/下压板位置(cm)、压板间隔(cm)      |

## 功能页面详细说明

### 页面 1：标准模式 (StandardModePage)

常规针刺生产模式。

| 参数         | 默认值 | 单位   |
| ------------ | ------ | ------ |
| 针刺速度     | 800    | 次/分  |
| 卷布速度     | 3.0    | cm/s   |
| 针刺深度     | 0.55   | cm     |
| 压板间隔     | 0.50   | cm     |
| 卷布长度     | 100.00 | m      |
| 手动卷布速度 | 8.33   | cm/s   |

**操作按钮**：针停高位、压板回零、开始、停止、长度清零、滚筒正转、速度写入、滚筒反转

---

### 页面 2：间歇模式 (IntermittentModePage)

间歇式针刺生产，支持分段参数配置。

- 主参数：针刺速度、程序号、间歇卷布速度、针刺深度、压板间隔
- 分段参数表（两组）：针刺长度 1/2、针刺速度 1/2、间歇长度 1/2
- 操作按钮：针停高位、压板回零、开始、停止

---

### 页面 3：渐变模式 (GradientModePage)

渐变密度针刺生产，支持「浅→深→浅」渐变模式。

- 参数：针刺速度、渐变/间隔卷布速度、针刺深度 1/2、压板间隔、渐变长度、间隔长度
- 模式标签「浅-深-浅」以青色高亮显示
- 操作按钮：针停高位、压板回零、开始、停止

---

### 页面 4：手动调试 (ManualDebugPage)

设备调试维护界面，可独立控制每个电机。

| 电机       | 单位    | 方向控制     |
| ---------- | ------- | ------------ |
| 针刺电机   | 转/分   | 正转 / 反转  |
| 卷布电机   | cm/s    | 正转 / 反转  |
| 上压板电机 | cm/min  | 上升 / 下降  |
| 下压板电机 | cm/min  | 上升 / 下降  |

每行均提供：速度输入框 + 加速 / 减速 / 写入按钮 + 方向控制按钮

---

### 页面 5：厂家设置 (FactorySettingsPage)

机械传动参数配置（齿轮比设置）。

| 参数       | 默认值 (A:B) |
| ---------- | ------------ |
| 针板速比   | 41 : 50      |
| 滚筒速比   | 1 : 10       |
| 上压板速比 | 1 : 5        |
| 下压板速比 | 1 : 5        |

附加参数：滚筒直径 82.00 mm、涡轮导程 0.4 mm / 0.3 mm

每行带「修改」按钮，底部「返回」按钮跳转回报警复位页。

---

### 页面 6：报警复位 (AlarmResetPage)

坐标校准与报警处理。

- **坐标显示**：针板坐标、上压板坐标、下压板坐标，各带「置零」按钮
- **回零速度**：针板 100 rpm、上压板 1500 rpm、下压板 1500 rpm
- **零位参数**：零位时上下压板间距 2.0 cm
- **快捷入口**：「齿轮比设置」按钮跳转到厂家设置页

底部提供三步校准操作指引：
1. 将下压板移动到目标位置后，将当前坐标置零
2. 设置上方压板间距参数，一般设为 2 或 3 即可
3. 根据压板零位间距值，将上压板移动到目标位置后，将上压板位置置零

## 界面设计风格

采用 **Precision Industrial Dark** 主题，具有现代化深色工业 HMI 风格：

- **背景**：深炭灰/石板色，带蓝色色调，模拟拉丝金属质感
- **强调色**：青色 / 蓝绿色 (`#3cc8f0`) 用于高亮和激活状态
- **输入框**：暗底亮字，底部带醒目蓝色边框，聚焦时边框发亮
- **按钮**：金属感渐变效果，配合 hover / pressed 交互反馈
- **颜色语义**：
  - 开始按钮 → 绿色调
  - 停止按钮 → 红色调
  - 写入/修改按钮 → 青色调
- **字体**：优先苹方 (PingFang SC)、Hiragino Sans GB 等中文优化字体
- **样式管理**：全部集中在 `MainWindow::globalStyleSheet()` 方法中，通过 Qt 属性选择器 `[cssClass="..."]` 实现语义化样式

## 开发说明

### 架构设计要点

- **页面管理**：`QStackedWidget` 承载所有页面，`SidePanel` 发送信号驱动切换
- **样式系统**：全局 QSS 集中管理，各页面组件通过 `setProperty("cssClass", "xxx")` 设置语义类名，无内联样式
- **跨页导航**：通过 Qt 信号槽机制实现（`returnRequested`、`gearSettingsRequested`）
- **MOC 处理**：CMake 的 `CMAKE_AUTOMOC` 自动处理 Qt 元对象编译
- **源码发现**：`file(GLOB_RECURSE ...)` 自动收集 `src/` 下所有源文件

### 如何新增页面

1. 在 `src/pages/` 下创建 `NewPage.h` 和 `NewPage.cpp`
2. 继承 `QWidget`，添加 `Q_OBJECT` 宏
3. 在构造函数中使用 `QGroupBox` + `QGridLayout` 组织布局
4. 在 `MainWindow.cpp` 中 `#include` 新页面并 `addWidget` 到 `m_stack`
5. 在 `SidePanel.cpp` 的 `names` 列表中添加对应按钮名称
6. 重新构建即可

### 如何自定义主题

编辑 `MainWindow::globalStyleSheet()` 中返回的 QSS 字符串即可。所有颜色、字号、边框、渐变均在此集中定义，修改后重新编译即刻生效。
